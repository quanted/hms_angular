About

This page visualizes the data retrieved after running an HMS simulation. Various graphs are displayed, as well as a table with the data, in an editable containerized fashion.

Properties

Property Description
Left Panel Located on the left of the page and accessed via an arrow button. Expandable left panel that contains a clickable list of the available data from the simulation and a mini map of the simulation location. Selecting data from the list or map will add the data to every container.
Containers Configurable containers that can be used to display data from the simulation.
Add Button Located on the top right of the page. Adds a new container to the page.

Containers

Included on this page are a number of configurable containers. These containers can be configured with different graphs or a table. Containers can be added or removed from the page and are drag/droppable as well.

Container Properties

Property Description
Menu Located at the top left of the container. Expands and allows for selection of chart type, and the ability to remove a container.
Drag Handle Located at the top right of the container. Allows for the container to be dragged to a different position.
Graph Located in the center of the container and visualizes selected data. Includes tools to analyze the data at the top of the graph under the title when hovered over. Selection of different graphs can be done by accessing the menu. Changing selected data can be done in the left panel or the legend panel.
Legend Button Located on the right center of the container. Expands and collapses the legend panel. Allows for the selection of different data to be displayed in the graph.
Table Located in the center of the container and visualizes selected data.
Export Button Located in the menu for the container and only displayed when the table is. Exports the data in the table to a .csv file.
Paginator Located in the bottom right of the container and only displayed when the table is. Displays a selected number of rows per page on the table.
